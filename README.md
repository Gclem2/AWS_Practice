# AWS Practice Repository

This repository contains hands-on exercises, configs, notes, and practice labs focused on learning and improving AWS skills. It includes examples, experiments, and testing environments used to better understand AWS services and architecture concepts.

## What's Inside

- Practice with core AWS services (EC2, S3, IAM, VPC, Lambda, RDS, DynamoDB, etc.)
- Small labs and experiments
- YAML and CLI configuration examples
- Notes and references
- Architecture diagrams and scenarios

## Purpose

To serve as a personal space for AWS practice, experimentation, and skill-building.  
Perfect for learning, studying, or refreshing AWS knowledge.

## Tools Used

- AWS CLI  
- CloudFormation / YAML  
- Linux shell environment  
- AWS Free Tier or sandbox account
